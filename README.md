# Code Review Agent

## Description

The Code Review Agent is designed to take in a GitHub repository and perform a comprehensive review. The current iteration focuses on the following aspects:

### Technical Details

- **Programming Language**
- **Frameworks**
- **Azure Services**
- **Design Patterns**
- **Project Type**

### Scores

- **Code Complexity Score:** 1 to 10
- **Reusability Score:** 1 to 10

### Business Insights

- **Industry**
- **Business Outcome/Value:** 
- **Target Audience:** Who is this repository aimed at?

## Core Logic

1. Download the zip file from github
2. Extract the repo in a local folder, performing various checks and sanitation measures
3. Iterate over all the files in the repo, excluding certain file extensions and files over 1MB. 
4. Print the files
5. Clean up the local folder

## Requirements

- **Python 3.x**
- **Libraries:**
  - `requests`
  - `python-dotenv`
